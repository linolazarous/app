# Demo Video Placeholder

To add your demo video:

1. Name your video file: `demo-video.mp4`
2. Place it in this folder: `/frontend/public/`
3. The video will automatically appear in the "Watch Demo" modal

Recommended video specs:
- Format: MP4 (H.264)
- Resolution: 1920x1080 (1080p)
- Duration: 1-3 minutes
- File size: Under 50MB for fast loading

The video should showcase:
- Creating a new project
- AI code generation in action
- One-click deployment
- Final deployed application
